# HTML Artifacts Examples

This document contains example prompts and expected HTML artifacts that demonstrate the HTML Artifacts feature.

## Example 1: Simple Interactive Button

**Prompt:**
```
Create a simple interactive button that changes color when clicked
```

**Expected Artifact:**
- Title: "Interactive Color Button"
- Description: "A button that cycles through different colors when clicked"
- HTML with:
  - Centered button with styling
  - JavaScript to change background color on click
  - Smooth color transitions

## Testing the Feature

To test the HTML Artifacts feature, you can:

1. Enable the "Artifact" toolkit in your chat preferences
2. Use any of the example prompts above
3. Verify that the artifact:
   - Renders correctly in the Preview tab
   - Shows the HTML code in the Code tab
   - Can be copied to clipboard
   - Can be downloaded as an .html file
   - Works interactively (if applicable)

## Tips for Creating Artifacts

- Be specific about what you want
- Mention if you want to use a specific library (Chart.js, D3.js, etc.)
- Provide data if you need visualizations
- Specify design preferences (colors, layout, style)
- Request specific interactive features
